# Aprimorium aplicado ao Projeto

## Função

Criar um starter reutilizável para APIs em Go.

## Forma

Estrutura modular:

- Handler
- Service
- Repository
- Config
- Response

## Fluxo

HTTP
→ Handler
→ Service
→ Repository
→ Database

## Resultado Esperado

Maior velocidade para iniciar projetos e aprofundar o domínio de Go.
